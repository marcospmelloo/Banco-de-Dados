# 📚 Banco de Dados

Este repositório reúne exercícios e atividades desenvolvidos durante meus estudos de **Banco de Dados**, principalmente ao longo da **Residência em TIC – Software do Serratec**.

A proposta deste repositório é registrar minha evolução na área de banco de dados por meio do desenvolvimento de um **banco de dados de uma biblioteca**.

Todos os exercícios estão baseados em um único projeto prático, onde foram aplicados conceitos de criação, manipulação e consulta de dados em SQL. Ao longo das atividades, o banco foi sendo construído e evoluído, simulando um sistema real de gerenciamento de biblioteca.

---

# 🎯 Objetivo do Repositório

Este repositório foi criado com alguns objetivos claros:

* Registrar minha evolução durante a **Residência Serratec**
* Organizar os exercícios de banco de dados desenvolvidos ao longo das aulas
* Servir como material de revisão para estudos futuros
* Demonstrar meu conhecimento em modelagem e manipulação de dados com SQL

Os exercícios não são isolados: todos fazem parte de um mesmo contexto, utilizando um banco de dados de biblioteca como base. Isso permite uma melhor compreensão prática de como os comandos SQL são aplicados em situações reais.

Além disso, ele também funciona como parte do meu **portfólio de aprendizado** na área de tecnologia.

---

# 🧠 Conteúdos Trabalhados

Todos os conceitos abaixo foram aplicados dentro de um banco de dados único, representando um sistema de biblioteca, onde foram realizadas diversas operações ao longo das atividades.

Os exercícios presentes neste repositório exploram conceitos fundamentais de banco de dados, como:

### DDL (Data Definition Language)

* Criação de banco de dados
* Criação de tabelas
* Definição de chaves primárias
* Definição de chaves estrangeiras
* Uso de restrições (NOT NULL, UNIQUE)

### DML (Data Manipulation Language)

* Inserção de dados (`INSERT`)
* Atualização de dados (`UPDATE`)
* Remoção de dados (`DELETE`)

### DQL (Data Query Language)

* Consultas com `SELECT`
* Filtragem de dados (`WHERE`)
* Organização de resultados

# 🗂 Estrutura do Banco

O projeto é baseado em um banco de dados de biblioteca, composto por tabelas principais como:

* **livro** – armazena informações sobre os livros
* **usuario** – registra os usuários do sistema
* **emprestimo** – controla os empréstimos realizados

As tabelas se relacionam entre si, permitindo simular operações reais como controle de empréstimos, consultas e organização dos dados.

# 📈 Processo de Aprendizado

Durante a residência do **Serratec**, os conteúdos de banco de dados são apresentados de forma progressiva. Inicialmente, são abordados conceitos básicos como criação de tabelas e definição de atributos. Em seguida, os exercícios evoluem para incluir relacionamentos entre entidades, integridade dos dados e alterações estruturais.

Este repositório acompanha essa evolução por meio de um projeto único de banco de dados de biblioteca. Ao longo das atividades, foram realizadas desde a criação das tabelas até consultas mais elaboradas, utilizando funções de agregação, junções e agrupamentos, simulando cenários reais de uso.
